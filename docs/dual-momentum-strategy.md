# 二八轮动策略（Dual Momentum Strategy）

## 策略概述

"蛋卷斗牛二八轮动"是一个经典的低波动轮动策略，通过动态切换不同市场指数来追求稳健收益。

## 策略原理

### 成分标的
- 沪深300指数（代表大盘股）
- 中证500指数（代表中小盘股）
- 国债指数（避险资产）

### 轮动规则

1. **回看周期**：20个交易日
2. **比较逻辑**：
   - 计算沪深300和中证500在过去20个交易日的涨跌幅
   - 选择涨幅较大的指数持有
3. **防守机制**：
   - 如果两个指数均下跌，则切换为持有国债指数
4. **调仓时机**：
   - 每日收盘后判断
   - 次日收盘时执行切换

## 策略特点

### 优势
- **趋势跟踪**：自动选择强势市场
- **风险控制**：熊市自动切换到债券避险
- **低波动**：通过轮动降低组合波动率
- **简单透明**：规则清晰，易于执行

### 适用场景
- 追求稳健收益的投资者
- 希望降低回撤的长期投资
- 不想频繁操作的懒人投资

## 实现要点

### 数据需求
- 沪深300日线数据（000300.SH）
- 中证500日线数据（000905.SH）
- 国债指数日线数据（H11025.CSI 或其他）

### 计算步骤
1. 获取当前交易日和20个交易日前的收盘价
2. 计算涨跌幅：`(当前价 - 20日前价) / 20日前价`
3. 判断持仓：
   - 若两者均上涨，选涨幅大者
   - 若一涨一跌，选上涨者
   - 若两者均下跌，选国债

### 回测指标
- 年化收益率
- 最大回撤
- 夏普比率
- 换手率
- 胜率

## 策略变种

可以考虑的优化方向：
- 调整回看周期（10日、30日等）
- 增加其他指数（创业板、科创50等）
- 添加止损/止盈规则
- 结合其他技术指标过滤信号
