# 宏观经济分析笔记

## 2026年3月11日

### 输入性通胀与贵金属价格关系

**核心观点：**

长期输入性通胀将会导致降息预期的下降，促使短期利率维持高位。聪明钱从贵金属资产出来后购买美元、美债、原油商品，从而导致贵金属价格下跌📉。

**传导机制：**

1. 输入性通胀持续 → 降息预期下降
2. 短期利率维持高位
3. 资金流向：贵金属 → 美元/美债/原油
4. 结果：贵金属价格承压下跌

### 费雪公式在宏观分析中的应用

**费雪交易方程式：MV = PT**

- M = 货币供应量（市面上有多少钱）
- V = 货币流通速度（钱一年转手几次）
- P = 物价水平
- T = 商品交易总量 / 总产出

**分析意义：**

作为宏观分析的基础工具，费雪公式帮助理解：
- 货币供应与物价的关系
- 流通速度对经济活跃度的影响
- 通胀/通缩的形成机制
- 货币政策的传导效果

**实际应用场景：**

1. 判断通胀压力：M↑ + V↑ → P↑（通胀）
2. 分析流动性陷阱：M↑ 但 V↓ → P不变（资金空转）
3. 评估经济活力：V的变化反映市场信心
4. 预测政策效果：央行调控M时需考虑V和T的变化
