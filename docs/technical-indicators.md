# 技术指标与量化评估指标

## 一、技术指标分类

### 1. 趋势指标 (Trend Indicators)

#### MA - 移动平均线
- **公式**: MA = (P1 + P2 + ... + Pn) / n
- **用途**: 平滑价格波动，识别趋势方向
- **常用周期**: 5日、10日、20日、60日、120日、250日
- **交易信号**: 
  - 金叉：短期MA上穿长期MA → 买入
  - 死叉：短期MA下穿长期MA → 卖出

#### EMA - 指数移动平均线
- **公式**: EMA(t) = α × P(t) + (1-α) × EMA(t-1)，其中 α = 2/(n+1)
- **优势**: 对近期价格更敏感，反应更快
- **用途**: 短线交易，快速捕捉趋势变化

#### MACD - 平滑异同移动平均线
- **公式**: 
  - DIF = EMA(12) - EMA(26)
  - DEA = EMA(DIF, 9)
  - MACD柱 = 2 × (DIF - DEA)
- **交易信号**:
  - DIF上穿DEA → 买入
  - DIF下穿DEA → 卖出
  - 柱状图由负转正 → 趋势转强
  - 背离：价格创新高但MACD不创新高 → 顶背离，看跌

#### 布林带 (Bollinger Bands)
- **公式**:
  - 中轨 = MA(20)
  - 上轨 = 中轨 + 2×标准差
  - 下轨 = 中轨 - 2×标准差
- **用途**: 判断超买超卖，波动率分析
- **交易信号**:
  - 价格触及上轨 → 超买，可能回调
  - 价格触及下轨 → 超卖，可能反弹
  - 布林带收窄 → 波动率降低，可能突破

---

### 2. 动量指标 (Momentum Indicators)

#### RSI - 相对强弱指标
- **公式**: RSI = 100 - [100 / (1 + RS)]，RS = 平均涨幅 / 平均跌幅
- **范围**: 0-100
- **交易信号**:
  - RSI > 70 → 超买，可能回调
  - RSI < 30 → 超卖，可能反弹
  - RSI背离 → 趋势可能反转

#### KDJ - 随机指标
- **公式**:
  - RSV = (收盘价 - N日最低价) / (N日最高价 - N日最低价) × 100
  - K = RSV的M1日移动平均
  - D = K的M2日移动平均
  - J = 3K - 2D
- **参数**: 通常N=9, M1=3, M2=3
- **交易信号**:
  - K > 80, D > 80 → 超买
  - K < 20, D < 20 → 超卖
  - K上穿D → 买入
  - K下穿D → 卖出

#### Momentum - 动量指标
- **公式**: MOM = 当前价格 - N日前价格
- **用途**: 衡量价格变化速度
- **交易信号**: 
  - MOM > 0 且上升 → 上涨动能增强
  - MOM < 0 且下降 → 下跌动能增强

---

### 3. 成交量指标 (Volume Indicators)

#### OBV - 能量潮指标
- **公式**: 
  - 若今日收盘价 > 昨日收盘价，OBV = 昨日OBV + 今日成交量
  - 若今日收盘价 < 昨日收盘价，OBV = 昨日OBV - 今日成交量
- **用途**: 通过成交量变化预测价格趋势
- **交易信号**:
  - 价格上涨，OBV上升 → 趋势健康
  - 价格上涨，OBV下降 → 量价背离，警惕

#### VWAP - 成交量加权平均价
- **公式**: VWAP = Σ(价格 × 成交量) / Σ成交量
- **用途**: 判断当日交易的平均成本
- **交易信号**:
  - 价格 > VWAP → 多头占优
  - 价格 < VWAP → 空头占优

---

### 4. 波动率指标 (Volatility Indicators)

#### ATR - 平均真实波幅
- **公式**: 
  - TR = max(最高价-最低价, |最高价-昨收|, |最低价-昨收|)
  - ATR = TR的N日移动平均
- **用途**: 
  - 衡量市场波动性
  - 设置止损位：止损 = 入场价 ± 2×ATR
  - 判断趋势强度

#### Historical Volatility - 历史波动率
- **公式**: σ = std(log returns) × √252
- **用途**: 
  - 期权定价
  - 风险管理
  - 仓位管理

## 二、量化策略评估指标

### 1. 收益类指标

#### 总收益率 (Total Return)
```
总收益率 = (期末资产 - 期初资产) / 期初资产 × 100%
```

#### 年化收益率 (Annualized Return)
```
年化收益率 = (1 + 总收益率)^(252/交易天数) - 1
```

#### 超额收益 (Alpha)
```
Alpha = 策略收益率 - 基准收益率
```

---

### 2. 风险类指标

#### 最大回撤 (Maximum Drawdown)
```
最大回撤 = (谷值 - 峰值) / 峰值
```
- **含义**: 从历史最高点到最低点的最大跌幅
- **用途**: 衡量策略的最大风险暴露

#### 波动率 (Volatility)
```
波动率 = std(日收益率) × √252
```
- **含义**: 收益率的标准差，年化处理
- **用途**: 衡量收益的不确定性

#### VaR - 风险价值
```
VaR(95%) = 资产价值 × 收益率的5%分位数
```
- **含义**: 在95%置信度下，未来一段时间内可能的最大损失

---

### 3. 风险调整收益指标

#### 夏普比率 (Sharpe Ratio)
```
Sharpe = (策略收益率 - 无风险利率) / 策略波动率
```
- **含义**: 每承担一单位风险获得的超额收益
- **评判**: 
  - Sharpe > 2: 优秀
  - Sharpe > 1: 良好
  - Sharpe < 1: 一般

#### 索提诺比率 (Sortino Ratio)
```
Sortino = (策略收益率 - 无风险利率) / 下行波动率
```
- **优势**: 只考虑下行风险，更合理

#### 卡玛比率 (Calmar Ratio)
```
Calmar = 年化收益率 / 最大回撤
```
- **含义**: 收益与最大回撤的比值
- **评判**: Calmar > 1 较好

#### 信息比率 (Information Ratio)
```
IR = (策略收益 - 基准收益) / 跟踪误差
```
- **含义**: 主动管理能力的衡量

---

### 4. 交易效率指标

#### 胜率 (Win Rate)
```
胜率 = 盈利交易次数 / 总交易次数
```

#### 盈亏比 (Profit/Loss Ratio)
```
盈亏比 = 平均盈利 / 平均亏损
```

#### 期望收益 (Expected Return)
```
期望收益 = 胜率 × 平均盈利 - (1-胜率) × 平均亏损
```

#### 换手率 (Turnover Rate)
```
换手率 = 交易金额 / 平均持仓市值
```
- **用途**: 衡量交易频率和成本

---

### 5. 其他重要指标

#### Beta系数
```
Beta = Cov(策略收益, 市场收益) / Var(市场收益)
```
- **含义**: 策略相对市场的系统性风险
- **解读**:
  - Beta = 1: 与市场同步
  - Beta > 1: 波动大于市场
  - Beta < 1: 波动小于市场

#### 持仓时间分布
- 平均持仓天数
- 最长/最短持仓时间
- 用途：判断策略类型（日内/短线/中长线）

#### 资金曲线特征
- 连续盈利/亏损次数
- 回撤恢复时间
- 收益曲线平滑度

## 三、指标组合使用建议

### 趋势跟踪策略
- 主要指标：MA、MACD、布林带
- 辅助指标：ATR（止损）、成交量
- 评估指标：Sharpe、最大回撤、Calmar

### 均值回归策略
- 主要指标：RSI、KDJ、布林带
- 辅助指标：波动率、成交量
- 评估指标：胜率、盈亏比、期望收益

### 动量策略
- 主要指标：Momentum、RSI、MACD
- 辅助指标：成交量、换手率
- 评估指标：Alpha、IR、Beta

### 多因子选股
- 技术因子：RSI、MACD、成交量
- 基本面因子：PE、PB、ROE
- 评估指标：IC、IR、多空收益

---

## 四、实战注意事项

1. **避免过度拟合**: 不要在同一数据集上反复优化参数
2. **样本外测试**: 必须在未见过的数据上验证策略
3. **交易成本**: 考虑手续费、滑点、冲击成本
4. **数据质量**: 注意复权、停牌、ST等特殊情况
5. **风险控制**: 单笔止损、总仓位控制、分散投资
6. **参数稳健性**: 测试参数在不同市场环境下的表现
7. **指标组合**: 多个指标相互验证，避免单一指标误判
